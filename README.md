# msgb
Message Box -- cli password (or other sensitive information) storage tool

# HOW TO USE:
[Optional] Place the tool (msgb.bat and src/) in a dir that is listed in the PATH variable.
    C:\> msgb addUser
    >> Username: Example
    >> Password:
    >> Retype Password
    >> Password:
    
    C:\>msgb new-rsa
    ...
    >> Username: Example
    >> Password:
    ...
    
    C:\> msgb addMsg [KEY] [MSG]
    >> Username: Example
    >> Password:
    >> Entry 1 [KEY: [KEY]] added.
